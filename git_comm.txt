$ git config --global user.email "tyeror2@bk.ru"	 ----------   Установить почту для пользователя


$ git init	--------	Создать локальную директорию


$ git remote add origin https://github.com/vlad-uchiha/first_repo	---------	Связать локальную директорию с удаленной 


$ git clone https://github.com/vlad-uchiha/first_repo	---------	Создать копию удаленной директории на компьютере (т.е. "скачать" ее на компьютер)


Коммит - сохранение конкретной версии файла внутри Git

$ git status	---------	Просмотреть какие файлы закоммичены


$ git add <NameOfFile> / $ git add .	---------	Подготавливает данные файлы к следующему коммиту, добавляя их в "индекс" гита.
													"Индекс" гита - хранилище файлов, готовых к коммиту
